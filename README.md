# Official Web Portfolio (Astro + Tailwind)

Premium, fast personal portfolio inspired by brittanychiang.com, mattfarley.ca, lauren-waller.com/work, olaolu.dev, gavinnegron.dev.

## Stack
- Astro
- Tailwind CSS
- Motion One (optional)

## Scripts
- dev: start local server
- build: generate static site
- preview: preview production build

## Getting started
1. Install deps
2. Run dev server

## Content
- Edit `src/pages/index.astro` and components in `src/components`
- Replace `public/resume.pdf` with your CV
- Add your `public/profile.jpg`

## Deploy
- Deploy to Vercel/Netlify; update `site` in `astro.config.mjs`
